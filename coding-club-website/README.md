# Coding Club Website

Welcome to the Coding Club website! This project serves as a platform for members to connect, share resources, and stay updated on upcoming events.

## Project Structure

The project is organized as follows:

```
coding-club-website
├── public
│   ├── index.html          # Homepage introducing the Coding Club
│   ├── events.html         # Information about upcoming events
│   ├── resources.html      # Resources for members
│   ├── members.html        # Member profiles and testimonials
|   |── register.html       # Students registering portal
│   └── assets
│       ├── css
│       │   ├── styles.css  # Main CSS styles
│       │   └── responsive.css # Responsive styles
│       ├── js
│       │   └── scripts.js   # JavaScript for interactivity
|       |   └──register.js   # register portal
│       └── fonts           # Custom fonts
├── src
│   ├── components
│   │   ├── header.html     # Header component
│   │   ├── footer.html     # Footer component
│   │   └── navbar.html     # Navigation bar component
│   ├── styles
│   │   ├── variables.css   # CSS variables
│   │   └── mixins.css      # Reusable CSS mixins
│   └── scripts
│       └── main.js         # Main JavaScript entry point
├── package.json            # npm configuration file
├── .gitignore              # Files to ignore in version control
└── README.md               # Project documentation
```

## Features

- **Homepage**: An introduction to the Coding Club with links to other pages.
- **Events Page**: Stay updated on upcoming events organized by the club.
- **Resources Page**: Access tutorials, articles, and tools to enhance your coding skills.
- **Members Page**: Read testimonials and learn about the contributions of club members.
- **Register Page**: Register students for club.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd coding-club-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open the `public/index.html` file in your browser to view the homepage.
